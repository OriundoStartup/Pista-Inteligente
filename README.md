# MVP de Análisis Hípico para Detección de Patrones Repetidos

Este proyecto es un Producto Mínimo Viable (MVP) diseñado para analizar resultados de carreras hípicas y detectar patrones de llegada (trifectas) que se repiten en el tiempo.

## Descripción
El sistema simula datos históricos de carreras (con capacidad futura de scraping real) y utiliza análisis de datos para identificar combinaciones de llegada (1°-2°-3°) que han ocurrido más de una vez.

## Stack Tecnológico
*   **Lenguaje:** Python
*   **Análisis de Datos:** Pandas
*   **Web Scraping (Esqueleto):** requests, BeautifulSoup4
*   **Visualización en Terminal:** rich

## Ejecución
Para probar la simulación y el análisis:

```bash
python analisis.py
```
